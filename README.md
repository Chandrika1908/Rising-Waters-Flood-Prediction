# 🌊 Rising Waters - Flood Prediction using Machine Learning

## 📌 Overview

Rising Waters is a Machine Learning based Flood Prediction System developed during the Foundry AI Internship.

The application predicts flood occurrence using historical rainfall and meteorological parameters. A trained Decision Tree model is integrated with a Flask web application to provide real-time flood predictions.
